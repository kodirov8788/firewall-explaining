# Firewall Explaining

An educational site designed to explain networking and firewall concepts through a visual, web-friendly presentation.

**Live demo:** https://firewall-explaining.vercel.app

## What This Project Shows
- Turning technical infrastructure topics into accessible product content
- Building an educational interface around explanation rather than CRUD
- Lightweight modern frontend architecture for shareable technical content

## Why This Stack / Tooling
- **Next.js** was chosen for SEO and easy deployment of explanatory content.
- **Framer Motion** adds clarity to transitions and visual storytelling where movement helps understanding.
- **Tailwind CSS** keeps the UI system compact and iteration-friendly.
- **TypeScript** helps maintain correctness even in a smaller educational app.

## Stack
- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion

## Key Features
- Educational content around firewall concepts
- Responsive, shareable frontend delivery
- Visual storytelling with motion support

## Getting Started
```bash
npm install
npm run dev
```
